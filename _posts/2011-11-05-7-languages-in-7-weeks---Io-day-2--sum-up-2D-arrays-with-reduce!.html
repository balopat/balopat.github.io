3. Write a program to add up all of the numbers in a two-dimensional array.

How it works? List has a reduce method:

Also known as foldl or inject. Combines values in target starting on the left. If no initial value is passed the head of the list is used. 

Io> list(1, 2, 3) reduce(+)==> 6Io> list(1, 2, 3) reduce(xs, x, xs + x)==> 6Io> list(1, 2, 3) reduce(+, -6) // Passing the initial value.==> 0Io> list(1, 2, 3) reduce(xs, x, xs + x, -6)==> 0
